# 智能购车咨询系统 - 项目说明

## 项目概述

基于 Vue3 + Element Plus 的智能购车意向咨询系统前端实现。

## 技术栈

- **框架**: Vue 3.5.22 (Composition API)
- **UI组件库**: Element Plus 2.11.7
- **状态管理**: Pinia 3.0.3
- **路由**: Vue Router 4.6.3
- **构建工具**: Vite 7.1.11
- **图标**: @element-plus/icons-vue 2.3.2

## 项目结构

```
car-project/
```

## 使用说明

### 用户登录
1. 访问登录页
2. 选择"用户登录"或"管理员"
3. 输入用户名和密码（暂时为模拟登录，任意用户名密码即可）

**测试账号**:
- 用户: 任意用户名/密码（选择"用户登录"）
- 管理员: 任意用户名/密码（选择"管理员"）

### 用户注册
1. 点击"立即注册"
2. 完成三步注册流程：
   - 账号信息
   - 个人信息
   - 购车偏好

### 新建咨询
1. 登录后在主页点击"开始新咨询"
2. 填写咨询表单：
   - 咨询标题
   - 购车预算
   - 偏好车型
   - 使用场景
   - 燃料类型
   - 品牌偏好
   - 详细需求
   - 选择AI模型
3. 提交后查看AI分析结果
4. 可以评分并保存咨询记录

### 查看咨询历史
1. 在侧边栏选择"咨询历史"
2. 支持搜索和筛选
3. 可查看详细信息

### 管理员功能
1. 以管理员身份登录
2. 查看数据概览
3. 管理所有咨询记录
4. 管理用户账号
5. 查看数据统计
